# Sudoku Solver

A Java program that solves Sudoku puzzles using the Backtracking Algorithm.

## Features
- Solves incomplete Sudoku puzzles automatically
- Validates rows, columns, and 3x3 grids
- Uses recursion and backtracking
- Displays the solved puzzle

## Technology Used
- Java

## How to Run
1. Run the Java application
2. View the solved Sudoku in the console
